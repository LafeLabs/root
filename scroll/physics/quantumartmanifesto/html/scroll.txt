<h2>Quantum VRML(QVRML)</h2>
<ul>
    <li>One qubit, two arbitrary VRML objects, fade and rotate and grow and shrink</li>
    <li>Two Qubits, four arbitrary VRML objects fade, rotate grow and shrink</li>
    <li>Two Qubits, one arbitrary VRML object rotate fade, grow and shrink(map to S^3)</li>
    <li>Arbitrary numbers of qubits from 2 to a few dozen, as a chain of objects which rotate along theta and phi angles, hence mapping to various theta's of S^N and phase angles phi.  Each link is some arbitrary VRML object, whole thing is manipulated with gestures, which controls qubit states.  Hence this is a full hand-gesture VRML interface which can link to multi qubit system in real time at video rate in a web browser</li>
</ul>
